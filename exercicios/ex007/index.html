Formatação de textos

Texto em negrito

Esse termo é em negrito para escrever qualquer coisa em negrito coloque o texto entre <b>TEXTO</b>

Nessa frase tem um termo importante usando a tag <strong>TEXTO</strong>

Itálico / Ênfase

Este termo está em Itálico para colocar um termo em Itálico coloque o texto entr <i>TEXTO</i>

Tenho um termo com Ênfase na minha frase, para escever ele digite o texto entre <em>TEXTO</em>

Texto marcado

Tenho um texto marcado para escrever ele digite o texto entre <mark>TEXTO </mark>

Texto pequeno

Podemos ter um texto pequeno para escrever ele digite <small>texto</small>

Texto deletado

Podemos ter um texto deletado para escrevermos ele temos que colocar o texto entre <del>TEXTO

Texto inserido

Um Texto inserido é um texto sublinhado, para sublinhar seu texto coloque ele entre <ins>TEXTO</ins>

Texto subescrito

Com texto subescrito podemos fazer coisas como H2O para escrever o subescrito coloque o termo entre <sub>texto</sub>

Texto sobescrito

Com texto sobescrito

Podemos escrever coisas como X2 + X = ? para escrever sobescrito coloque o termo entre <sup>termo</sup>
